##### ParseInt

```go
// 把一个字符串转为整型

// base：字符串的进制数，如果该参数是0，那么根据字符串的前缀来判断进制，
// 2 for "0b", 8 for "0" or "0o", 16 for "0x", and 10 otherwise。
// 比如：s是10；base为2时，结果是2；base为10时，结果是10。

// bitSize：限定了结果的取值范围。0，8，16，32，64 分别对应 int, int8, int16, int32, and int64。
// 如果bitSize < 0 || bitSize > 64，一个错误会被返回。
// 当结果值超过该参数设定的范围时，一个错误会被返回。

// 返回结果：err 是一个接口值，具体类型是 *NumError，NumError的具体字段参考包中的定义。
// 如果s为空或是包含无效的数字，i = 0，err.(*NumError).Err = ErrSyntax
// 如果s的值超出了取值范围，i = 取值范围内最大的正负值，err.(*NumError).Err = ErrRange
func ParseInt(s string, base int, bitSize int) (i int64, err error) {}
```
